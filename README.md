# webs-obsport-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文档形式存储，不针对任何特定网站或域名，仅作为内容存档与展示用途。

## 项目简介

`webs-obsport-notes` 是一个轻量级、无依赖的静态 HTML 页面集合。仓库内的每个 HTML 文件均为独立的完整页面，可直接在浏览器中打开查看。项目不涉及后端服务、数据库或动态内容生成，适合作为纯前端资料的存档与分发。

## 目录说明

仓库根目录下的 HTML 文件即为可访问的页面。建议按如下方式组织文件：

- 首页或入口文件可直接命名为 `index.html`（如有）
- 其余页面以有意义的英文或拼音命名，例如 `note-about.html`、`archive-2024.html`
- 可创建子目录存放相关资源（如 CSS、图片、脚本等），但非必需

示例结构：

```
/
├── index.html
├── page-one.html
├── page-two.html
└── assets/
    ├── style.css
    └── logo.png
```

## 页面归档说明

- 每个 HTML 页面均为自包含文件，可在本地或任意静态托管服务上直接运行。
- 页面内容以记录、归档、展示为目的，不包含交互式后端逻辑。
- 所有页面均无外部依赖（除非在 HTML 中显式引用 CDN 资源）。
- 本仓库不收集、不跟踪任何用户数据。

## 维护说明

- 欢迎提交 Pull Request 新增或更新 HTML 页面。
- 请确保新增页面为独立、完整的 HTML 文档（包含 `<!DOCTYPE html>`、`<html>`、`<head>`、`<body>` 等基本结构）。
- 建议在页面头部添加 `<meta charset="UTF-8">` 和 `<title>` 标签。
- 如引用外部资源，请优先使用相对路径或可靠的 CDN 链接。
- 请勿在页面中包含任何恶意代码、跟踪脚本或违反适用法律的内容。
- 仓库维护者保留合并或拒绝任何提交的权利。
